# Paper Networks Audit — narration

Voice: Kokoro `af_heart` (local default).  
Regenerate a line with:

```bash
export HYPERFRAMES_PYTHON=~/.venvs/hf-media/bin/python
npx hyperframes tts "<text>" -v af_heart -o assets/voice/<id>.wav
```

| id | Scene | Text |
|----|-------|------|
| intro | Intro | Paper Networks Audit helps teams uncover suspicious clusters of manuscripts, linked by shared IP, email, device, or Word metadata. |
| clusters | Explore | Start by exploring the network. Filter by connection type and journal, then focus on the clusters with the most overlapping signals. |
| compare | Compare | Open a cluster to compare its papers side by side. Shared attributes light up, so suspicious patterns are obvious at a glance. |
| tracking | Collaborate | Every review action is tracked across the team. Who has seen, checked, resolved, or flagged each paper, always kept in sync. |
| flags | Export | Flags build a complete audit trail. Export to C S V, or share a manager summary with the research integrity team. |
| outro | Outro | The whole review state is stored in the URL, so any finding is one link away. [long pause] Paper Networks Audit, from signal to decision. |

### Outro pause

The outro WAV is built as two clips with **1.2s** of silence between them (the `[long pause]` marker is not spoken).
